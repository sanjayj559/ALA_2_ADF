# 📱 API Explorer Flutter App

## 📌 Project Overview

This project is developed as part of Android Development ALA. The application integrates a public REST API, fetches JSON data, parses it, and displays it using GridView in Flutter.

## 🚀 Features

* Fetch data from REST API
* JSON parsing
* Display data using GridView
* Image loading from internet
* Responsive UI

## 🛠️ Technologies Used

* Flutter
* Dart
* HTTP Package
* REST API

## 🌐 API Used

https://jsonplaceholder.typicode.com/photos

## ▶️ How to Run

1. Install Flutter SDK
2. Clone the repository:
   git clone https://github.com/your-username/api_explorer_app.git
3. Navigate to project:
   cd api_explorer_app
4. Install dependencies:
   flutter pub get
5. Run the app:
   flutter run
